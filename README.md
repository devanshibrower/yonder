# Yonder

A website showcasing major meteor shower schedules.

## Tech Stack

- Next.js
- Typescript
- Tailwind CSS

## Getting Started

1. Clone the repo

```bash
git clone https://github.com/devanshibrower/yonder.git
cd yonder
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm run dev
```

4. Open http://localhost:3000
