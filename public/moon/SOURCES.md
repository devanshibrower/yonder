# Moon Asset Sources

- `nasa-lroc-color-2k.jpg`: NASA Scientific Visualization Studio CGI Moon Kit, LRO/LROC color map, https://svs.gsfc.nasa.gov/4720/
- `nasa-moon-2026-samples.json`: six-hour samples derived from NASA Scientific Visualization Studio `mooninfo_2026.json`, https://svs.gsfc.nasa.gov/5587/. Used by SkyMoonStudy only — MoonSky computes this geometry instead (see `src/lib/moonGeometry.ts`).
- `svs-2026-3840/moon.4489.png`, `moon.4495.png`, `moon.4501.png`, `moon.4507.png`, `moon.4513.png`, `moon.4519.png`, `moon.4525.png`, `moon.4531.png`, `moon.4537.png`: local prototype subset converted from NASA SVS 2026 3840x2160 plain Moon TIFF frame set with alpha, https://svs.gsfc.nasa.gov/vis/a000000/a005500/a005587/frames/3840x2160_16x9_30p/plain/. Used by SkyMoonStudy only.
